"# FinalBackendV2" 

<h1>Proyecto de Defensa Final – UTIC</h1>

<h2>Carrera:</h2>
<p>Licenciatura en Análisis de Sistemas</p>

<h2>Descripción del Proyecto</h2>
<p>
    Este proyecto está desarrollado en <strong>C# (.NET Core 9)</strong> siguiendo el patrón 
    <strong>MVC (Model–View–Controller)</strong>.<br>
    Para el acceso a datos se utiliza <strong>ADO.NET</strong>, y la base de datos implementada es 
    <strong>PostgreSQL 16</strong>.
</p>
