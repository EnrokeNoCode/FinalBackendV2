"# FinalBackendV2" 

Proyecto de Defensa Final – UTIC

Carrera: Licenciatura en Análisis de Sistemas

Este proyecto está desarrollado en C# (.NET Core 9) siguiendo el patrón MVC (Model–View–Controller).
Para el acceso a datos se utiliza ADO.NET, y la base de datos implementada es PostgreSQL 16.
